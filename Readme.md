Getting Started 🚀

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

Prerequisites 📋

- Composer
- PostgreSQL
- Symfony-CLI

Installation 🔧

``composer install``

Executing tests ⚙️

``php bin/phpunit --filter test-name``
